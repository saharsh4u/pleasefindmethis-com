What to include in the request

Exact model and aliases

Exact identifiers reduce wrong leads and help search across marketplaces and old pages.

Required accessory bundle

List missing pieces early so finders do not submit a source that is correct but incomplete.

Condition limits

Small visual and condition differences often decide whether a source is useful or another near match.

Functional proof required

Tell finders what evidence makes a source acceptable before they submit a lead.

Import or proxy tolerance

Describe this constraint in plain language so finders can filter wrong leads before submission.

Common search failures

  • Wrong variant
  • Missing accessory
  • Untested listing
  • Fungus, haze, meter, or battery-door issues

Verification checks

  • Sample photos
  • Serial or reference
  • Lens and finder condition
  • Shutter and meter behavior
  • Seller reputation
