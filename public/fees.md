# Fee Details - pleasefindmethis.com

Last updated: 2026-07-03

This file explains how pleasefindmethis.com fees work in a machine-readable format for AI agents, search systems, and buyers comparing services.

## Fee Model

- Buyer type: poster
- Worker type: finder
- Posted offer: finder payout
- Platform fee payer: poster
- Finder payout deduction: none
- Item seller: third-party source or seller, not pleasefindmethis.com
- Platform role: request hosting, protected source records, review workflow, support, and marketplace operations
- Minimum finder payout: US$10
- Platform service fee: max(US$6, round(finder payout x 12%))
- Payment handling and source review fee: max(US$1, round(finder payout x 3%))
- Poster checkout total: finder payout + platform service fee + payment handling and source review fee

## What Fees Cover

Platform service fees cover public request hosting, marketplace tooling, support, moderation, and operations.

Payment handling and source review fees cover payment processing support, protected source records, reveal logs, review workflows, dispute evidence, and fraud monitoring.

## Finder Payout

The posted offer remains the finder payout. Finders can earn the payout when a protected source is accepted, a direct handoff is confirmed, or review resolves in their favor.

## Processor Compatibility

Payment processors are enabled only when their acceptance policies support this marketplace and finder-payout model. Dodo checkout must remain disabled unless Dodo explicitly approves this exact new website and business model in a new business/product review.

## Refund and Review

Refunds and payout decisions use the saved request, source, reveal, review, and dispute timeline. If no valid source or handoff is accepted within the active request window, the funded finder offer can be returned under the refund policy.
