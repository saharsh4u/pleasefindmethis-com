# Community Comment Bank

These are helpful-first drafts for Reddit, Discord, forums, and comment sections. Use the no-platform version by default. Add disclosure only where the destination rules allow a product/tool mention.

Disclosure options:

- `Disclosure: I built pleasefindmethis.com, so skip this if self-promo is not allowed. The useful part is the search method below.`
- `Disclosure: I work on pleasefindmethis.com. Sharing only because this is exactly the kind of search workflow we think about.`
- `If tools are allowed here, you could also post the image/request on pleasefindmethis.com for extra search help.`

## 25 Drafts

1. For exact product ID from a photo:
   Crop the image tightly around the item, then try Google Lens, Bing Visual Search, and Yandex Images. Search terms like `"[main color] [material] [shape] [item type]"` usually work better than brand guesses. Before buying, match at least two details: stitching, hardware, logo placement, label, dimensions, or connector shape.

2. For clothing from a screenshot:
   Describe the garment in order: cut, neckline, sleeve, fabric, pattern, and color. Example: `black ribbed square neck long sleeve top contrast stitching`. If it is from TikTok/Instagram, search the creator name plus `outfit`, `link`, `LTK`, or `ShopMy`.

3. For furniture:
   Search the distinctive parts, not the generic category. Try terms like `cantilever chrome frame tan leather sling chair` or `oak spindle back dining chair curved arms`. Verify joinery, screw placement, underside labels, and dimensions.

4. For discontinued items:
   Use Google date filters and terms like `site:ebay.com`, `site:poshmark.com`, `site:mercari.com`, `sold`, `archive`, `replacement`, and `model number`. The Wayback Machine can reveal exact SKUs and color names.

5. For duplicate Amazon-style listings:
   Reverse image search the main product photo, then search a unique phrase from the listing title in quotes. Verify by dimensions, material, and review photos instead of trusting the brand name.

6. For shoes/sneakers:
   Search visible design details: `white gum sole green heel tab mesh runner`. Check outsole pattern, heel tab, tongue label, and side panel shape before calling it a match.

7. For mystery cables:
   Identify both connector ends, pin count, shape, and embossed markings. Search `proprietary 6 pin round connector power adapter` or `USB C to barrel jack 5.5 2.1mm`. Match voltage, polarity, and amperage before buying.

8. For replacement parts:
   Search the product model number first, not the part description. Try `"[model number]" "replacement" [part name]`. If there is no listing, search for donor units and compatible assemblies.

9. For lamps or fixtures:
   Search by shade shape, mount type, finish, and number of bulbs: `opal globe brass flush mount 3 light`. For safety, check UL/ETL certification before installing electrical fixtures.

10. For vintage or thrifted items:
    Check maker's marks, tags, zipper brands, country of origin, and care labels. Search `vintage [brand] [tag color] [material] [item]`. For value, use sold listings, not active listings.

11. For posters/art prints:
    Reverse image search first, then add `artist`, `print`, `poster`, `gallery`, `edition`, or visible text. If marketplaces are the only results, try to locate the original artist before buying.

12. For movie/TV wardrobe:
    Search show name + character + episode + item type. Fan wikis, ShopYourTV, Spotern, Reddit episode threads, and costume forums often beat general search.

13. For cheaper source:
    Once you know the exact item, search the product image and the first 6-8 words of the title in quotes. Also search the SKU if available. Compare shipping, return policy, and seller reputation.

14. For scam shops:
    Search the domain plus `reviews`, `scam`, `Trustpilot`, and the exact product image. Check address, refund policy, and contact email. If everything is 60-80% off and the domain is new, treat it as high risk.

15. For plush or toy:
    Use species, pose, fabric texture, eye style, tag color, and size: `brown floppy dog plush red bow 12 inch`. Try eBay sold listings, WorthPoint, Mercari, and collector groups.

16. For jewelry:
    Search by stone cut, setting, metal color, clasp type, and motif. Ask for hallmarks, weight, close-up photos, and return terms before buying anything expensive.

17. For wallpaper/tile:
    Search visual descriptors plus repeat pattern: `blue white chinoiserie peel stick wallpaper birds`, `zellige green 4x4 glossy tile`. Request a sample first because color and scale can shift.

18. For bags/backpacks:
    Look for zipper pulls, strap attachment, logo placement, pocket layout, and lining. For authentication, compare hardware engravings and interior labels, not just the outside logo.

19. For cosmetics/skincare:
    Zoom into cap color, bottle shape, pump style, and label layout. Search packaging terms like `amber glass dropper white label green text serum`. Verify shade names and ingredients on the brand site.

20. For books with partial covers:
    Search visible title fragments in quotes plus `book cover`. If only art is visible, describe it: `novel cover red dress woman stairs gold title`. Try WorldCat, Goodreads image search, and Google Books.

21. For obscure hardware:
    Search by function plus shape: `spring loaded clamp with threaded knob`, `stainless bracket offset slotted holes`. Include measurements. Trade forums often know the name retailers do not use.

22. For anime/game merch:
    Search character name + item type + manufacturer terms like `official`, `acrylic stand`, `plush`, `figure`, `ichiban kuji`, `prize`. Check MyFigureCollection or official shops to avoid bootlegs.

23. For original image source:
    Use reverse image search across Google, Yandex, Bing, and TinEye. Sort by oldest result if possible. Add `before:YYYY-MM-DD` in Google if you know when it started circulating.

24. For asking a community:
    Include the image, country, budget, size, where you saw it, and what you already tried. A useful title is specific: `Looking for this green ribbed glass pendant light, likely EU/UK, under GBP200`.

25. For close-enough alternatives:
    List non-negotiables first: color, size, material, budget, country, and what can vary. Search with `dupe`, `similar`, `alternative`, `inspired by`, and `lookalike`. Verify reviews with real customer photos.
