# $1,000 Paid Acquisition Test

Do not spend until the live domain, HTTPS, request flow, checkout, thank-you state, and tracking events are verified.

## Test Shape

- Duration: 14 days
- Geography: US English first
- Device: mobile-first
- Primary paid KPI: funded bounty CPA
- Early lead KPI if checkout volume is low: qualified request CPA
- Qualified request definition: user uploads/pastes an item reference, selects category, enters budget/reward, and provides email/account.

## Kill And Scale Rules

Kill the launch if tracking, checkout, or thank-you conversion fails.

Pause a keyword/ad set after `2x target CPA` spend with zero qualified requests.

Pause if CTR is below the threshold after 1,000 impressions:

- Google Search: 3%
- Reddit Ads: 0.35%
- TikTok: 0.6%
- Pinterest: 0.4%
- Meta retargeting: 1%

Scale only after 3+ qualified requests and CPA under target. Increase budget 20-25% every 48 hours.

Reallocate remaining budget in this order:

1. Google exact-match winners.
2. Best TikTok/Reels Spark video.
3. Pinterest keyword clusters with saves and outbound clicks.
4. Reddit ads only if comments are positive and CPA is stable.
