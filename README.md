# pleasefindmethis.com

[pleasefindmethis.com](https://pleasefindmethis.com/) is a free public request board for exact, discontinued, sentimental, sold-out, and otherwise hard-to-find items.

When image search or marketplace search returns lookalikes, a requester can publish one structured page with reference photos, must-match details, failed searches, wrong matches, buying constraints, and proof requirements. Signed-in helpers can add public source links, seller paths, model clues, compatibility notes, search terms, and safety warnings.

## How the product works

1. Describe the exact item and select the closest request category.
2. Add up to four reference photos and the details that distinguish the item from near matches.
3. Publish the request free and share its public URL.
4. Review public clues from signed-in helpers.
5. Independently verify any third-party seller before buying elsewhere.

Pleasefindmethis is not a marketplace, personal shopper, escrow service, or seller-verification service. It does not collect rewards, hold funds, pay helpers, broker purchases, or sell the requested items.

## Stack

- React 19 and TypeScript
- Vite
- Node.js server routes
- Supabase authentication, database, and storage
- Vercel deployment and analytics

## Local development

Requirements: Node.js 24 and npm.

```bash
npm install
npm run dev
```

Create a local environment file with the required Supabase and authentication values before testing account, publishing, or notification flows. Never commit credentials.

Build and test:

```bash
npm run build
npm test
```

## Safety and reports

Public clues are suggestions, not verified listings. Requesters remain responsible for checking availability, price, seller identity, authenticity, compatibility, website safety, and payment terms.

Report abuse, unsafe content, privacy concerns, or security issues to [support@pleasefindmethis.com](mailto:support@pleasefindmethis.com). Please do not include secrets or sensitive personal data in a public issue.
